## 📊 Features

### 🔗 URL Shortening  
- Generates unique short codes  
- Supports optional custom codes  
- Validates URL format  

### 📈 Analytics  
- Tracks click count  
- Logs timestamp + referrer  
- Displays analytics in a stats page  

### 🧾 Dashboard  
- Lists all shortened URLs  
- Shows click totals  
- Links to stats + QR code  

### 🧩 QR Codes  
- Generates QR codes for each short URL  
- Opens as PNG image  

### 🎨 UI  
- Clean, modern interface  
- Responsive layout  
- Copy-to-clipboard button  

---

## 🧠 How It Works

1. User enters a long URL  
2. App validates it  
3. A unique short code is generated  
4. Data is stored in SQLite  
5. Redirect route logs analytics  
6. Stats page displays click history
